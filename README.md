# 基于P4-RF的DDoS实时攻击检测与防御

#### 介绍
随着网络安全威胁日益复杂和频繁，传统的DDoS攻击检测与防御方法已经显露出一定的局限性。针对这一挑战，结合可编程数据平面（P4）技术和机器学习中的随机森林（RF）算法，构建基于P4-RF的DDoS实时攻击检测与防御系统具有重要的理论意义和实际应用价值。
